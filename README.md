# MNIST Handwritten Digit Classification

## Overview
This project implements a Convolutional Neural Network (CNN) to classify handwritten digits from the MNIST dataset. It follows a structured, bilingual (English & Persian) approach as detailed in the project conventions.

## Project Structure
- `notebooks/`: Contains the main Jupyter Notebook.
- `outputs/`: Stores generated plots and models.
- `requirements.txt`: Python dependencies.

## Setup
To set up the environment, we recommend using `uv`:

```bash
uv venv
source .venv/bin/activate  # Or .venv\Scripts\activate on Windows
uv pip install -r requirements.txt
```

## How to Run
Launch the notebook using Jupyter:
```bash
jupyter notebook notebooks/mnist_classification.ipynb
```
